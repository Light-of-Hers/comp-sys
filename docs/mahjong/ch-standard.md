References:

- 《中国麻将竞赛规则》 1998 年国家体育总局
- [wiki]([https://zh.m.wikipedia.org/zh-hans/%E5%9B%BD%E6%A0%87%E9%BA%BB%E5%B0%86](https://zh.m.wikipedia.org/zh-hans/国标麻将))

### 得分

- 底分 = 8
- 自摸 = (底分 + 番数) x 3
- 被放铳 = 底分 x 3 + 番数 （放铳者承担番种分数）

### 番数

- 不重复原则
- 不拆移原则
- 就高不就低原则
- 不得相同原则
- 套算一次原则
